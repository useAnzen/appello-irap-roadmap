1 · Objective

agentc2/tests-e2e/fixtures/auth.fixture.ts already replicates user envs; appello/plan/ documents the construction flows; Fathom MCP captures sessions structurally. This epic is Nathan's structured field-test program: 6 sessions with real crews on 3 customer sites, each session running through 5 canonical flows on real iPads, recorded via Fathom so we have video + transcript baselines for the same flows the Spider Swarm tests synthetically.

2 · Acceptance Criteria

  • 6 field sessions scheduled across 3 customer sites (Owens Insulation, Clearstory, third TBD).
  • Per session: 5 canonical flows (clock-in, timesheet, work-order add, signature, photo-upload) run by 2 crew members.
  • Recordings + transcripts in Fathom tagged field-test:<site>:<date>.
  • Per-flow metrics captured: task time, error count, friction notes, qualitative quotes.
  • Findings synthesized into appello/plan/field-test-report-2026Q2.md with: top-5 friction points, recommended fixes, comparison to Spider synthetic baselines (C1-0010).
  • Crew consent forms signed; PII scrubbed before Fathom indexing.
  • Crew thanked + compensated per Appello research-incentive policy.
  • ≥ 2 confirmed product changes from findings shipped within 60 days post-report.

3 · Architecture & Approach

Session protocol

Per session (90 min):
  10 min - context + consent
  60 min - flows (5 × 12 min budget per flow)
  10 min - debrief + open feedback
  10 min - artifact handoff (photos, quotes, friction notes)
All recorded in Fathom (camera on crew tablet over shoulder + screen)

4 · Implementation Phases (35h)

  1. Site coordination + consent + scheduling (6h).
  2. Protocol authoring + facilitator practice (4h).
  3. 6 sessions (18h on-site).
  4. Synthesis + comparison to synthetic (5h).
  5. Report + shareable summary (2h).

5 · Risks & Mitigations

  • Crew availability slips. Mitigation: 3-site spread = no single dependency.
  • Hawthorne effect (crews perform better when watched). Mitigation: baseline metrics from production analytics also reported.
  • PII exposure in recordings. Mitigation: consent + scrub + indexed only after legal review.
