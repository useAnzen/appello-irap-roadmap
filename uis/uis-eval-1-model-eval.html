1 · Objective

Establish defensible model selection before agents are wired up. Evaluate GPT-4o, Claude 3.5 Sonnet, Gemini 1.5, and Llama 3 on 50 real Appello TypeScript code samples. Score for code comprehension, fix accuracy, and cost per query. Select primary model + fallback.

2 · Acceptance Criteria

  • 50 representative Appello code samples curated (mix of frontend React, backend Express, GraphQL resolvers, Prisma schema).
  • Scoring rubric defined (comprehension, fix accuracy, cost-per-call, latency).
  • All 4 candidate models run end-to-end; scores recorded in a reproducible eval harness.
  • Decision memo published with primary + fallback selection and rationale.
  • Per-call cost telemetry plumbed into CostEvent so future model swaps are measurable.

3 · Architecture & Approach

Stand-alone eval harness in `agentc2/packages/agentc2/src/evals/` that loads samples from `appello-sdlc/eval-set/`, hits 4 model APIs via the same provider abstraction used at runtime, and writes results to Supabase for comparison. Output → markdown decision memo committed to `appello-sdlc/docs/`.
