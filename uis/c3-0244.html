1 · Objective

agentc2/CLAUDE.md is the onboarding doc; agentc2/docs/ + appello-roadmap/plans/ are the working surface; agentc2/apps/agent/src/app/pulse/[slug]/manage/page.tsx is the team backlog view. This epic is NH2 onboarding — the same 5-day / 4-week pattern as C1-0018 + C1-0020 + C1-0045 (NH1 series), adapted for C3 / ATLAS work as NH2's primary focus area.

2 · Acceptance Criteria

  • 5-day plan published at agentc2/docs/onboarding/NH2-week-1.md, focused on ATLAS + RAG + Spider testing context (where NH2 will be productive).
  • Day 1: local dev bootstrap (re-uses C1-0013).
  • Day 2: ATLAS deep-dive — read ATLAS-MCP-SETUP.md, walk the rag-tools, query the existing collection.
  • Day 3: Spider Swarm overview — run 1 spec end-to-end, view in dashboard.
  • Day 4: appello-roadmap/plans/ tour + Pulse manage page.
  • Day 5: tiny P2 hamburger in C3 area (e.g. add a regex to area-classifier) → merged PR.
  • 4-week ramp: 1 P2 (week 1), 1 P2 + RAG ingest (week 2), 1 P2 + Spider spec (week 3), 1 P1 paired with Emma (week 4).
  • Buddy: Ian 30 min Mon/Wed/Fri for month 1.
  • Sign-off: NH2 has +1 reviewer privilege on agentc2 PRs touching ATLAS or Spider areas.
