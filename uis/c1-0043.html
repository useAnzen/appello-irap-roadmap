1 · Objective

AgentC2 already exposes run/status endpoints (agent-operations-tools.ts, AgentRun queries); appello-sdlc/apps/frontend/src/lib/actions/router/actions.ts proxies them to the frontend. This epic builds the React Query / SWR hooks layer on top so any UI component can subscribe to live stage outputs and run status with a one-liner — no per-component polling logic.

2 · Acceptance Criteria

  • Hook useAgentRuns({ workPackageId?, agentSlug?, status? }) returns paginated AgentRun rows with live updates.
  • Hook useAgentRun(runId) returns single run + tail of stage outputs via SSE.
  • Hook useAgentStatus(agentSlug) returns current load (in-flight runs / queued / mean latency).
  • Hooks built on React Query; SSE subscription multiplexed via the C1-0005 channel so we have one server connection per tab.
  • All hooks return { data, isLoading, error, refetch }; optimistic-update API for mutations.
  • Stale-while-revalidate default 30s; opt-out per call.
  • TypeScript types generated from the OpenAPI of C1-0002 — no hand-written types.
  • Storybook examples + docs page covering common patterns.

3 · Architecture & Approach

Hook layer

// apps/frontend/src/hooks/agent-runs.ts
export function useAgentRuns(filter: AgentRunFilter) {
  return useQuery({
    queryKey: ['agent-runs', filter],
    queryFn: () => actions.listAgentRuns(filter),
    staleTime: 30_000,
  });
}

export function useAgentRun(runId: string) {
  const qc = useQueryClient();
  useEffect(() => {
    const off = liveStream.subscribe(`agent-run:${runId}`, (msg) => {
      qc.setQueryData(['agent-run', runId], (old: any) => ({
        ...old, stageOutputs: [...(old?.stageOutputs ?? []), msg]
      }));
    });
    return off;
  }, [runId]);
  return useQuery({ queryKey: ['agent-run', runId], queryFn: () => actions.getAgentRun(runId) });
}

5 · Risks & Mitigations

  • Hook re-render storms on chatty agents. Mitigation: coalesce SSE updates per 250ms frame.
  • OpenAPI codegen drift. Mitigation: codegen runs in CI; PR fails if types stale.
  • Cache invalidation bugs. Mitigation: typed invalidation keys; React Query devtools enforced in dev.
