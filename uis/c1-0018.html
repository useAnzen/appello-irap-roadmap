1 · Objective

The canonical onboarding surface already exists: agentc2/CLAUDE.md, agentc2/docs/sdlc/, appello-sdlc/README.md, and claude-for-appello/skills/. This epic packages it into a guided NH1 walkthrough: a sequenced 5-day plan that delivers a first merged PR on day 5, working through (a) the agent framework, (b) the SDLC pipeline, (c) the appello-sdlc dashboard, (d) the Cursor IDE skill set, and (e) the hamburger factory end-to-end.

2 · Acceptance Criteria

  • 5-day plan published at agentc2/docs/onboarding/NH1-week-1.md; each day has explicit reading list + hands-on task + success criterion.
  • Day 1: AgentC2 architecture (read CLAUDE.md + 00-overview.md); deliverable = local-dev bootstrap success per C1-0013.
  • Day 2: SDLC pipeline (read 02–04 docs); deliverable = trigger a uis-generator run end-to-end on a sample idea.
  • Day 3: appello-sdlc dashboard (read README + walk screenshots); deliverable = add a small UI tweak end-to-end (PR opened, reviewer trust score reviewed).
  • Day 4: Cursor skills (read claude-for-appello/skills/); deliverable = run one skill (agentc2-feature-forge) successfully.
  • Day 5: Hamburger factory (read 03-hamburger-factory.md); deliverable = a tiny P2 hamburger goes from manifest entry → merged PR.
  • Walkthrough validated by Ian sitting through it himself before NH1 start; estimated total < 30 hours hands-on for new hire.

4 · Implementation Phases (32h)

  1. Curate reading list per day (5h).
  2. Author hands-on task brief per day (8h).
  3. Set up sample idea + sample UI tweak + sample P2 hamburger (8h).
  4. Self-walkthrough by Ian + timing pass (8h).
  5. Final doc + buddy-checkin schedule (3h).

5 · Risks & Mitigations

  • Docs go stale between authoring and NH1 start. Mitigation: CI link-checker on the walkthrough doc; Ian re-runs day 1+5 the week before NH1 start.
  • Day-5 P2 hamburger blocked by an outage. Mitigation: 2 backup hamburgers pre-selected; safe-rollback documented.
  • NH1 stuck on Day 2 (uis-generator setup). Mitigation: recorded screen-capture of the happy path embedded in the doc.
