📎 Evidence of how Appello ships software

NRC reviewers can see Appello's live development cadence on the public Appello Product Roadmap — Release 1.33 (LEM Foundation, T&M Work Orders, Equipment Module, Forms) currently has 45 PRs in QA and 17 ready-to-build items, each with a corresponding uis-*.html implementation spec. This IRAP project builds the AI development infrastructure (C1 SDLC Flywheel + C2 Spider Swarm) that will accelerate this cadence ~10×, and the C3 ATLAS / C4 Apex stack that exposes Appello's data as an AI-native customer experience.
